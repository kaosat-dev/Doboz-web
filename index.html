Doboz by kaosat-dev

Software tools for reprap/repstrap control

Dependencies

Jquery
Jquery Ui
Jquery Ui notify
Absolution: Jquery Ui Theme
Pyserial
Bottle
Tornado

Install

- download and unzip/untar the files
- launch the server using python: test.py
- open a browser to localhost:8000/index.html to use the interface and enjoy :)
